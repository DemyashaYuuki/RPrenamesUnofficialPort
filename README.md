# RP Renames Fabric Port

Это компактный порт идеи **RP Renames** под Fabric **1.21.10 / 1.21.11** на новой системе item models без старого CIT.

Что делает эта версия:
- перехватывает загруженные `item assets` клиента;
- рекурсивно проходит по `select / condition / range_dispatch / composite` item models;
- вытаскивает варианты, завязанные на `minecraft:custom_name`;
- показывает найденные варианты для предмета в первом слоте наковальни;
- позволяет кликнуть по варианту, чтобы сразу подставить имя.

Это **не байт-в-байт копия** полного upstream GUI из последнего RP Renames, а компактная реализация ядра под новую систему переименований.

## Сборка локально

По умолчанию проект настроен на **1.21.11**.

```bash
gradle build
```

Готовый jar появится в `build/libs/`.

## GitHub Actions

В репозитории уже есть workflow `.github/workflows/build.yml`, который собирает матрицу для:

- **1.21.11**
